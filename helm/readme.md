  helm upgrade demo42 ./helm/importantThings --reuse-values

** Dogfood **
```sh
az acr task create \
  -n cmd-helm \
  --file ./helm/acr-task.yaml \
  -c https://github.com/AzureCR/cmd.git \
  --git-access-token ${GIT_TOKEN} \
  -r commands
```
